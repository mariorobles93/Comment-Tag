Create Comments Service for Task Card Objective: Develop a Comments Service that integrates with the Task Card feature to manage user comments. This service will allow users to add, retrieve, update, and delete comments associated with specific task cards.
Requirements: A list of comments A field that can allow people to add new comments The field should be able to detect when you type in someone's name starting with an @, similar to the example in Slack. Here is a static set of users [ {'userID' : 1, 'name' : 'Kevin'}, {'userID' : 2, 'name' : 'Jeff'}, {'userID' : 3, 'name' : 'Bryan'}, {'userID' : 4, 'name' : 'Gabbey'}, ] When the entry is entered, it needs to detect which user was typed in and trigger a JavaScript function that alerts the person who is tagged.
